
# BlueKernel

A simple decription of BlueKernel and BlueOs 

BlueKernel is a kernel made in C using the bare bones tutorial of OSDev wiki

## Features

- Print text
- Colored text and Background
- Add lines
- Auto scroll
- Multiboot
- Custom made Library

If you want to run BlueKernel you need to install the dependencies needed to build the iso. You can find them in the OSDev wiki bare bones tutorial





## Deployment

To Build BlueKernel

```bash
  ./Build_BlueOS.sh
```
